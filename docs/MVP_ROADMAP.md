# ShellCompany MVP Roadmap

This roadmap outlines the steps to transform ShellCompany from a prototype into a real, functional multi-agent desktop app.

---

## Phase 1: MVP (Weeks 1-2)
- Implement backend models, APIs, agent runner, and logging
- Build basic React UI for Board Room, Agents, Console, Projects
- Manual agent/task assignment and execution
- Real-time log streaming (WebSocket planned)

## Phase 2: Multi-Agent Network (Weeks 3-4)
- Support launching multiple agent processes
- Add agent registration/configuration UI
- Enable concurrent task execution
- Improve status dashboard

## Phase 3: Advanced Features (Weeks 5+)
- Integrate with real AI providers (OpenAI, Claude, Gemini, etc.)
- Add persistent project/task tracking (DB-backed)
- Implement monitoring and cost tracking
- Build demo mode for onboarding
- Polish UI/UX, add visual aids/screenshots

## Phase 4: Documentation & Community
- Update docs-site with real feature status, API reference, usage guides
- Add troubleshooting, FAQ, and contribution guidelines
- Encourage open source contributions
